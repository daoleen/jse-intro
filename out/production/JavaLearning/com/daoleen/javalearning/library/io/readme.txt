Все потоковые классы из этого пакета происходят из следующих абстрактных классов:
- InputStream   - определяет байтовый поток для чтения
- OutputStream  - определяет байтовый поток для записи
- Reader        - определяет символьный поток для чтения
- Writer        - определяет символьный поток для записи

FileInputStream - наследник InputSteam
