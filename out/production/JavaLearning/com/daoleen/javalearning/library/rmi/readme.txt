Дистанционный вызов методов (Remote Methods Invocation - RMI) - позволяет объектам Java, выполняющимся
на одном компьютере, вызывать методы объектов Java, выполняющихся на другом компьютере. Это важная возможность,
поскольку она позволяет создавать распределенные приложения.

В качестве примера реализовано клиент-серверное приложение, в котором сервер получает запрос от клиента, обрабатывает
его и возвращает результат. Запрос состоит из двух чисел. Сервер суммирует их и возвращает полученный результат.
